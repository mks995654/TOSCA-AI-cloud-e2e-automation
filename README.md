# Tosca AI + Cloud End-to-End Automation Framework

## Project Overview
This project demonstrates an enterprise-level, AI-driven test automation framework using Tricentis Tosca. 
It covers UI, API, database validation, CI/CD integration, and cloud execution.

## Tech Stack
- UI Automation: OpenCart Demo
- API Automation: DummyJSON
- Automation Tool: Tricentis Tosca
- CI/CD: Jenkins
- Cloud: AWS / Azure
- Version Control: GitHub

## Key Features
- Model-based automation using Tosca
- AI Scan for intelligent UI recognition
- Risk-based test optimization
- API and UI end-to-end validation
- Jenkins CI/CD pipeline integration
- Cloud-based parallel execution
